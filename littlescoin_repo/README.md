# ğŸ§¸ Littles Program Overview

The **Littles Program** is the heart of the project.  
Families and children in need are:

1. Verified manually  
2. Assigned a public profile  
3. Displayed transparently on the website  
4. Connected to their own receiving wallet  
5. Monitored for follow-up assistance  

This brings **visibility, dignity, connection and direct impact**.

Profiles include:

- Photo  
- Story / context  
- Wallet address  
- Status of assistance  
- Transaction history (on-chain)
